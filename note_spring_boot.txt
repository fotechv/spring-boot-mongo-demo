Sau đó chạy các lệnh
./gradlew --stop ⇒ Dừng daemon Gradle đang chạy nền, Giải phóng bộ nhớ, fix lỗi daemon bị treo
./gradlew --refresh-dependencies ⇒ Làm mới cache dependencies của Gradle:
./gradlew build ⇒ Build toàn bộ project (compile + test + package), Tạo file JAR/WAR trong thư mục build/libs/
./gradlew clean build ⇒ Xóa sạch thư mục build/ cũ
./gradlew dependencies --configuration runtimeClasspath ⇒ Hiển thị tất cả dependencies cần thiết khi chạy ứng dụng, 👉 Perfect để kiểm tra conflict phiên bản thư viện
